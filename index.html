MultiController for Mac

Hello this is a multicontroller for mac mostly for toontown

Current features:

  • Mirror Mode for 2 toons

Planned features:

  • Multi Mode for 2 toons
  • Make it not look like shit
  • Enable/Disabled keep on top
  • multiple groups
  • essentially make this a mac version of the current windows one
  • other stuff if people want i guess

Download: v0.0.1

How to use

  • Click the 'Left' button, then select the toontown window on the left, wait until the multi comes back and make sure it says the name of the toontown process
  • Same thing but with right button
  • Click the toggle button so it says active, it is now listening to your key presses, wow!
  • It is now broadcasting every keypress to both of your toontown processes

notes:
this isnt restricted to toontown, it will broadcast your keypresses to most windows. knowing this, dont use it to broadcast to random processes because idk what could happen lol
