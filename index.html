MultiController for Mac

Hello this is a multicontroller for mac mostly for toontown
Demonstration GIF

Current features:

  • Mirror Mode for 2 toons
  • Multi Mode for 2 toons

How to use

  • Click the 'Left' button, then select the toontown window on the left, wait until the multi comes back and make sure it says the name of the toontown process
  • Same thing but with right button
  • Click the toggle button to enable, it is now listening to your key presses, wow!
  • Press ` (grave/tilde key above tab) to switch between modes
    • Mirror Mode
      • Broadcasts every key press to both processes
    • Multi Mode
      • escape opens book on both toons
      • Left Toon:
        • Jump: Left Shift
        • Move: WASD
      • Right Toon:
        • Jump: Right Shift
        • Move: Arrows
